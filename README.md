# Smartech Analytics
Smartech Analytics empowers marketing-tech leaders with AI-driven insights from campaign and lead data. Using Gemini 3.0 Flash's analytical capabilities with AlloyDB's vector AI features for intelligent data exploration and visualization.

## Features
- Dashboard with key metrics
- AI-powered chart generation from natural language queries
- Save and retrieve custom reports
- Real-time data visualization using Chart.js

## Usage
- Visit the dashboard to see overview metrics
- Enter natural language queries like "Show lead conversion rates by campaign" to generate charts
- Save interesting reports for future reference
