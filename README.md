# Smartech Analytics
Smartech Analytics empowers marketing-tech leaders with AI-driven insights from campaign and lead data. Using Gemini 3.0 Flash's analytical capabilities with AlloyDB's vector AI features for intelligent data exploration and visualization.

## Problem Statement (PoC)
Marketing teams struggle to turn campaign and lead data into timely, actionable insights. Existing analytics tools often require manual query building, spreadsheet work, or static dashboards that don’t adapt to natural language input. As a result:
- campaign performance and lead conversion trends are hard to explore quickly,
- report generation is slow and fragmented,
- non-technical users cannot easily create visualizations from business questions,
- saved insights are difficult to preserve and reuse.

Smartech Analytics addresses these challenges by combining AI-powered chart generation, database-backed campaign and lead data, and reusable report storage to make marketing analytics faster, more intuitive, and more accessible.

## Features
- Dashboard with key metrics
- AI-powered chart generation from natural language queries
- Save and retrieve custom reports
- Real-time data visualization using Chart.js

## Usage
- Visit the dashboard to see overview metrics
- Enter natural language queries like "Show lead conversion rates by campaign" to generate charts
- Save interesting reports for future reference
